### Documentation for Amorest.

##### How to run?

- Run git clone
`git clone --single-branch -b docs https://github.com/RamaMahardika/amorest.git`

- Install docsify globaly
`npm i docsify -g`

- Select amorest directory
`cd amorest`

- Serve documentation folder
`docsify serve docs`

- go to `http://localhost:3000/#/`
